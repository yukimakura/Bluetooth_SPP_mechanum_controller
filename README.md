# Bluetooth_SPP_mechanum_controller
UARTから来る信号により、メカナムホイールを動作させる物。  
mechanummoverオブジェクト生成時に与えられている引数を変更することで、   
それぞれのピン番号を変更することができる。   
   
動作検証機材   
|Genre|NAME|
|:--|:--:|
|Arduino|Arduino  NANO(328p)|
|Bluetooth module|SBDBT5V|
|MOTORDRIVER|MD10C|
